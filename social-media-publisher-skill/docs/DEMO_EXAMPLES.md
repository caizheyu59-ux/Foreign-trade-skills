## 🎥 录制演示 GIF

### 使用工具

**推荐工具**：
- macOS: GIPHY Capture
- Windows: ScreenToGif
- Linux: Peek

### 录制步骤

1. **准备环境**
   - 清空浏览器缓存
   - 准备好视频和内容文件
   - 关闭无关标签页

2. **开始录制**
   - 打开录制工具
   - 设置录制区域（浏览器窗口）
   - 开始录制

3. **执行操作**
   - 运行上传脚本
   - 正常操作流程
   - 保持鼠标移动流畅

4. **后期处理**
   - 裁剪多余区域
   - 调整帧率（建议 10-15 fps）
   - 添加文字说明
   - 压缩文件大小
