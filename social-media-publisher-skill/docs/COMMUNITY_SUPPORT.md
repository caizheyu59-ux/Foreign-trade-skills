## 📚 贡献指南

### 代码贡献

1. **Fork 仓库**
   ```bash
   # 在 GitHub 上点击 Fork
   ```

2. **克隆仓库**
   ```bash
   git clone https://github.com/YOUR_USERNAME/social-media-publisher-skill.git
   cd social-media-publisher-skill
   ```

3. **创建分支**
   ```bash
   git checkout -b feature/your-feature-name
   ```

4. **提交代码**
   ```bash
   git add .
   git commit -m "feat: add your feature"
   git push origin feature/your-feature-name
   ```

5. **创建 Pull Request**
   - 在 GitHub 上创建 PR
   - 填写 PR 描述
   - 等待审核
