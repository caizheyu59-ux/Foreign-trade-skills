# 更新日志 (CHANGELOG)

## [2.0.0] - 2026-03-31

### ✨ 新增
- 重构为直接翻译模式，无需外部 API
- 支持 Excel 格式（.xlsx, .xlsm, .xls）
- 支持自定义翻译词库（JSON 格式）
- 内置长文本翻译模板
- 智能文本识别（自动跳过网址、邮箱、产品编号）

### 🛠️ 改进
- 优化文本识别逻辑
- 提升翻译准确性
- 简化使用流程

### 📚 文档
- 完善 README.md
- 添加 CONTRIBUTING.md
- 添加 CHANGELOG.md

## [1.0.0] - 2026-03-31

### ✨ 新增
- 初始版本发布

## 语义化版本

本项目遵循 [语义化版本 2.0.0](https://semver.org/lang/zh-CN/)：

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正
