## 📊 使用说明摘要

### 快速开始（5 分钟）

```bash
# 1. 安装依赖
pip install -r requirements.txt
playwright install chromium

# 2. 配置环境
cp .env.example .env
# 编辑 .env 填写账号信息

# 3. 检查配置
python scripts/check_config.py

# 4. 登录 LinkedIn
python scripts/cli.py login

# 5. 添加潜客
python scripts/cli.py add --url "URL" --name "姓名" --priority high

# 6. 开始监控
python scripts/cli.py check
```

### 1. 信息安全

- ✅ `.env.example` 是模板，不含真实账号
- ✅ `.gitignore` 已配置，敏感文件不会提交
- ⚠️ 不要手动添加 `.env` 或 `data/` 到 Git

### 2. 合规使用

- ⚠️ 遵守 LinkedIn 服务条款
- ⚠️ 建议使用专门的监控账号
