## 🔑 关键配置项

### WhatsApp 号码配置

**文件：** `config/monitored-products.json`

```json
{
  "notifications": {
    "whatsapp": {
      "target": "+86XXXXXXXXXXX"
    }
  }
}
```

### 监控商品配置

**文件：** `config/monitored-products.json`

```json
{
  "products": [
    {
      "id": "product-001",
      "name": "商品名称",
      "url": "https://detail.1688.com/offer/XXX.html",
      "target_price": 45.00,
      "notify_threshold": 5
    }
  ]
}
```

### 定时任务配置

**文件：** `HEARTBEAT.md`（OpenClaw 工作区根目录）

```markdown
## 1688 价格监控

**执行频率：** 每小时整点

**执行命令：**
```bash
cd skills/1688-price-monitor
node price-alert.js
```
```
